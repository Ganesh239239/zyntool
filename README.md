# iLoveIMG Clone

Free online image editing tools built with Cloudflare Pages & Workers.

## Features
- ✅ Compress IMAGE
- ✅ Resize IMAGE
- ✅ Crop IMAGE
- ✅ Rotate IMAGE
- ✅ Convert to/from JPG
- ✅ Watermark IMAGE
- ✅ Meme Generator
- ✅ Photo Editor
- ✅ Upscale Image
- ✅ Remove Background
- ✅ HTML to IMAGE

## Deployment
1. Push this repository to GitHub
2. Connect to Cloudflare Pages
3. Set build output directory: `public`
4. Deploy!

## Local Development
```bash
npm install
```

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Cloudflare Workers + Functions
- Image Processing: @cf-wasm/photon
